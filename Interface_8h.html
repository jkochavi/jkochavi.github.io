void task_CAN ( void *  p_params)

Task which interacts with the CAN bus.

This task sends the current user-chosen drive mode to the PLC, and reads the accumulator pressure, using the CAN bus. We interface with the CAN bus using an MCP2515 controller.

◆ task_HALL1()

void task_HALL1 ( void *  p_params)

Task which reads a hall effect sensor.

This task polls the GPIO pin for the hall effect sensor. Whenever the signal reads high, the task retains the elapsed time, in milliseconds, between pulses. We use the circumference of the wheel to calculate the distance traveled between pulses, and then divide by the time between pulses to calculate speed, which is subsequently converted to RPM.

Parameters
p_paramsA pointer to function parameters which we don't use.

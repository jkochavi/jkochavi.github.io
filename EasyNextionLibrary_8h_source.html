36  /* five main functions of the library:
37  *
38  * -- begin(): the begin() method of the class in which we pass the initialization data to the objects.
39  * initialization data: unsigned long baud = 9600 (default) if nothing written in the begin()
40  * myObject.begin(115200); for baud rate 115200
41  *
42  * -- EasyNex(HardwareSerial& serial): The constructor of the class that has the parameter of the Serial we use
43  * EasyNex.myObject(Serial); or Serial1, Serial2....
44  *
45  * -- writeNum(String, unsigned int): for writing in components' numeric attribute
46  * String = objectname.numericAttribute (example: "n0.val" or "n0.bco".....etc)
47  * unsigned int = value (example: 84)
48  * Syntax: | myObject.writeNum("n0.val", 765); | or | myObject.writeNum("n0.bco", 17531); |
49  * | set the value of numeric n0 to 765 | | set background color of n0 to 17531 (blue)|
50  *
51  *
52  * -- writeStr(String, String): for writing in components' text attributes
53  * String No1 = objectname.textAttribute (example: "t0.txt" or "b0.txt")
54  * String No2 = value (example: "Hello World")
55  * Syntax: | myObject.writeStr("t0.txt", "Hello World"); | or | myObject.writeNum("b0.txt", "Button0"); |
56  * | set the value of textbox t0 to "Hello World" | | set the text of button b0 to "Button0" |
57  *
58  * -- NextionListen(): It uses a custom protocol to identify commands from Nextion Touch Events
59  * For advanced users: You can modify the custom protocol to add new group commands.
60  * More info on custom protocol: https://www.seithan.com/ and on the documentation of the library
61  * WARNING: This function must be called repeatedly to response touch events
62  * from Nextion touch panel. Actually, you should place it in your loop function.
63  *
64  * -- readNumber(String): We use it to read the value of a components' numeric attribute
65  * In every component's numeric attribute (value, bco color, pco color...etc)
66  * String = objectname.numericAttribute (example: "n0.val", "n0.pco", "n0.bco"...etc)
67  * Syntax: | myObject.readNumber("n0.val"); | or | myObject.readNumber("b0.bco"); |
68  * | read the value of numeric n0 | | read the color number of the background of button b0 |
69  */
